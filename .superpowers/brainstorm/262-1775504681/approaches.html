Como conectar o Excel ao Dashboard HTML?

Escolha a abordagem que melhor se encaixa no seu fluxo de trabalho

A

Excel Manual → Script Python → HTML

Você preenche o Excel com os dados das visitas (pode copiar do Agendor). Um script Python lê o Excel e atualiza o dashboard. Você clica em um arquivo .bat para atualizar.

Vantagens

  • Simples e confiável
  • Total controle dos dados
  • Dashboard bonito e customizado

Desvantagens

  • Precisa instalar Python
  • Atualização manual (1 clique)

⭐ Recomendado — equilíbrio ideal entre simplicidade e poder

B

API do Agendor → HTML em tempo real

Um servidor local Python consulta o Agendor automaticamente e serve o dashboard atualizado. Os dados vêm direto do CRM, sem Excel.

Vantagens

  • Dados sempre atualizados
  • Zero entrada manual

Desvantagens

  • Precisa de API key do Agendor
  • Servidor precisa ficar rodando
  • Dependente da API
C

Excel → HTML puro (sem instalação)

Você exporta o Excel como .csv e o HTML lê o arquivo diretamente no navegador usando JavaScript. Sem Python, sem servidor.

Vantagens

  • Nenhuma instalação
  • Funciona em qualquer PC

Desvantagens

  • Navegador bloqueia leitura de arquivos locais
  • Precisa de servidor HTTP simples mesmo assim
  • Menos robusto
